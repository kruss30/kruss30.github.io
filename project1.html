PL/0 Compiler and Virtual Machine (C)

Description

In this project, I built major components of a small compiler toolchain in C for a simplified PL/0-style programming language. The project involved implementing a scanner to tokenize source code, a recursive descent parser to enforce grammar rules, and a symbol table to manage identifiers and scope.

The compiler generated instructions for a stack-based virtual machine, which I also implemented. This virtual machine executed the generated instruction set, allowing programs written in the language to run and produce output.

Key Accomplishments

  • Produced clear and descriptive syntax error messages with accurate line numbers.
  • Implemented a custom instruction set and stack machine execution model.

Skills Learned

  • Lexical analysis and tokenization
  • Recursive descent parsing techniques
  • Symbol table management and scope handling
  • Intermediate code generation
  • Implementing and debugging a stack-based virtual machine
