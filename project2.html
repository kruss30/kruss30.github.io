eCitadel Blue Team Competition (Windows Server 2022 Domain Controller)

Description

In the eCitadel Cyber Challenge, I worked primarily on the Windows Server 2022 machine named headman, which served as the Active Directory Domain Controller and hosted a scored DNS service. The goal was to investigate an active security incident, remove malicious activity, and harden systems while keeping critical services available.

Key Actions and Results

  • Identified and removed a Sliver C2 beacon on headman and blocked attacker traffic with firewall rules.
  • Audited users and permissions and removed unauthorized access to reduce risk and configuration drift.
  • Configured Windows audit logging to restore visibility into authentication and system activity.
  • Restored Windows Defender functionality and securely configured Windows Defender Firewall settings.
  • Hardened core services by securing DNS and RDP configurations and removing legacy components such as SMBv1.
  • Reviewed and secured Active Directory configuration and updated user passwords as part of remediation.

Skills Learned

  • Windows Server 2022 hardening and defensive configuration
  • Incident response triage and malware removal workflow
  • Windows logging and audit policy configuration
  • Firewall rule design for service uptime and containment
  • Active Directory and DNS security fundamentals in a scored environment
